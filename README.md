## Calculator CLI

Run the calculator with a basic arithmetic operation and two numbers:

```bash
npm start -- add 4 5
node src/calculator.js / 12 3
```

Supported operations:

- `add` or `+`
- `subtract` or `-`
- `multiply` or `*`
- `divide` or `/`
